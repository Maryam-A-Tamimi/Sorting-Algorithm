#  Sorting Algorithms Testing 

##  Overview
This project focuses on implementing and analyzing multiple sorting algorithms using C++.  
It evaluates their performance and correctness across different input scenarios to understand their efficiency and behavior.

---

##  Algorithms Implemented
- Insertion Sort  
- Heap Sort  
- Quick Sort  
- Counting Sort  

---

##  Testing & Analysis
- Tested algorithms using different input types:
  - Random data  
  - Sorted data  
  - Reverse-sorted data  

- Compared execution time and performance on large datasets  
- Verified correctness of sorting results  
- Analyzed behavior under different conditions
